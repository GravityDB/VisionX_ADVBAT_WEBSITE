Why use ADVBAT?



ADVBAT is an external tool or framework that is capable of defining new language constructs, like Classes, Objects, Functions, Modularity and more. ADVBAT is also compatibale with other plugins and frameworks, like batbox and many others. Some Key features of ADVBAT is mentioned below

  • - Modularity
  • - Variables and Direct Access Memory
  • - Classes, objects, Data Hiding, Encapsulation, Polymorphism, abstraction and Dynamically linked library
  • - Asynchronous and Multi-threading
  • - Shared Memory Applications
  • - Standard Library
  • - Package Management



Installing ADVBAT



Installing ADVBAT is very Easy! ADVBAT installer is a very simple CLI that has very little to do. It has a folder selector and that's it, nothing more. If you have'nt downloaded the package already, do it from here:

DOWNLOAD ADVBAT

The installation installs important files from the web as well, so the user must have enabled the internet before running the setup file. Run the setup file in ADMIN as it will need to update and Access system path variable (Global System Variables)
